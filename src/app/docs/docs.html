Avatar

An avatar is a personalized graphical illustration that represents a computer user, or a character or alter ego that represents that user.

Standard Avatars

The avatar component has 5 sizes:

avatar-xl (9rem) avatar-lg (7rem) avatar-std (5rem) avatar-sm (4rem), and avatar-xs (2rem) .

Badge

Badges are used to add additional information to any content like notification count or status information.

Badge on Avatars

The badges to represent the availability status:

badge--online (green color) badge--dnd (red color) badge--offline (gray color)

Badge on Icons

The badges to represent the count. You can use class : badge__count
